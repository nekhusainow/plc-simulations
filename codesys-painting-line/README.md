# Automated Painting Line Control System

## Описание
Демо-проект автоматизированной системы управления покрасочной линией,
реализованный в среде CODESYS 2.3.
Проект моделирует полный технологический цикл окраски деталей
с транспортировкой, поворотом изделия, нанесением краски,
контролем уровня краски и аварийной сигнализацией.

## Функциональность
- Управление подвесным конвейером (М1)
- Вертикальное перемещение форсунки (М2)
- Поворот детали на 90° (М3)
- Управление электромагнитным клапаном подачи краски (К1)
- Покраска фронтальных и боковых поверхностей
- Контроль крайних положений (0° / 90°)
- Контроль уровня краски в баке
- Счётчик окрашенных деталей
- Аварийная сигнализация
- Сброс аварий оператором
- Визуализация технологического процесса

## Реализация
- Язык программирования: Structured Text (ST)
- Логика реализована на основе состояний (STATE / STEP)
- Использованы таймеры TON для технологических операций
- Реализованы аварии:
  - отсутствие детали
  - ошибка поворота
  - низкий уровень краски
- Прерывистая сигнализация аварий
- Проект протестирован в режиме симуляции

## Инструменты
- CODESYS 2.3
- IEC 61131-3
- Симуляция без физического ПЛК
