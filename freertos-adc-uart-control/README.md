# FreeRTOS ADC Control UART (STM32)

## Описание
Демо-проект встроенной системы на базе STM32 с использованием FreeRTOS.
Реализовано управление задачей чтения АЦП по командам,
передаваемым через UART.

Проект демонстрирует работу с задачами FreeRTOS,
бинарными семафорами и периферией микроконтроллера.

## Архитектура системы
- RTOS: FreeRTOS
- MCU: STM32
- Периферия: ADC, UART
- Механизм синхронизации: Бинарный семафор
- Управление задачей через команды UART

## Функциональность
- Команды `ADC ON` / `ADC OFF` по UART
- Запуск и остановка задачи чтения ADC
- Непрерывное измерение при активном режиме
- Передача результатов измерений по UART
- Использование семафора для синхронизации задач

## Реализация
- Отдельная задача `AdcTask`
- Бинарный семафор для запуска измерений
- Циклическое чтение ADC с задержкой
- Расчёт напряжения и температуры
- Асинхронное управление через UART
