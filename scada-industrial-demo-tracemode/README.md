# Industrial SCADA System with PLC Emulation (TRACE MODE 6)

## Описание
Демо-проект SCADA-системы для автоматизированного участка
термообработки, хранения и дозирования продукта.
Проект реализован в среде TRACE MODE 6
с использованием эмуляции PLC и технологического объекта.

## Функциональность
- Многоэкранный HMI оператора
- Навигация между технологическими участками
- Отображение аналоговых и дискретных сигналов
- Тренды технологических параметров
- Архивация данных
- Журнал событий и тревог
- Настройка параметров регулятора
- Имитация технологического объекта

## Реализация
- SCADA: TRACE MODE 6
- Языки: FBD, ST
- Использован PC-based контроллер
- Реализована эмуляция PLC и объекта управления
- Обмен данными по Modbus (в режиме симуляции)
- Реализована запись данных в MS Access (SQL)
